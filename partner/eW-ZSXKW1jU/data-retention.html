Deletion Procedures

User-initiated deletion

  • User requests account deletion via app Settings → "Delete Account"
  • System immediately revokes all financial institution access tokens
  • All PII and financial data purged within 30 days of request
  • Consent records retained for 7 years as required for legal compliance
  • Confirmation email sent to user upon completion

Inactive account deletion

  • Accounts inactive for 3 years are flagged for deletion
  • User notified by email 60 and 30 days before scheduled deletion
  • Deletion proceeds if no response is received

Financial access token revocation

When an account is deleted or a bank connection is removed, the access token is immediately revoked with the banking data partner via API and deleted from our encrypted storage. The token is not recoverable after revocation.

Data Export

Users may request an export of their personal data by contacting privacy@mintwarden.com. Exports are provided in JSON format within 30 days of the request at no charge.

Security of Data at Rest

  • All data containing PII or financial information is encrypted at rest
  • Financial access tokens are encrypted using strong symmetric encryption with key derivation — never stored in plain text
  • Storage systems require authenticated access with audit logging enabled

Policy Review

This policy is reviewed and updated annually, or whenever relevant laws or partner compliance requirements change.
